SHELL INTEGRATION
================================================================================

For zsh (~/.zshrc):
  export CLAUDE_REGISTRY_PATH="$HOME/.claude/registry.json"
  alias cpm="claude-pm"
  alias cpp="claude-pm select"
  
  cpc() {
    local project_path
    project_path=$(claude-pm select "$@")
    if [[ -n "${project_path}" ]]; then
      cd "${project_path}"
    fi
  }

For bash (~/.bashrc):
