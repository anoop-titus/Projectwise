<task type="auto">
  <name>Task 4: Implement claude-status command</name>
  <files>~/.zshrc (add function)</files>
  <action>
    Add claude-status() function to .zshrc:

    Behavior:
    1. Read .registry.json
    2. Calculate statistics:
       - Total projects: count all
       - Active projects: count status="active"
       - Favorite projects: count favorite=true
       - Total sessions: sum of session_count
       - Last accessed: most recent last_accessed timestamp

    3. Display with gum style:
       ```
       📊 Claude Project Workflow Status

       Projects: {total}
       ├─ Active: {active_count}
       ├─ Favorites: {favorite_count}
       └─ Archived: {archived_count}

       Usage:
       ├─ Total sessions: {session_count}
       ├─ Last project: {last_accessed_project} ({time_ago})
       └─ Last activity: {time_ago}

       Paths:
       ├─ Registry: {path to .registry.json}
       ├─ Active: {count in /active/}
       └─ Favorites: {count in /favorites/}
       ```

    4. Optional flag:
       - --detailed: Show all projects sorted by recent
       - --json: Output as JSON for parsing

    Usage:
      claude-status           # Show brief status
      claude-status --detailed    # Show all projects
      claude-status --json    # JSON output
  </action>
  <verify>
    claude-status
    # Should display project stats

    claude-status --detailed
    # Should show all projects list

    claude-status --json
    # Should output valid JSON
  </verify>
