  <name>Task 1: Enhance registry-update.sh with atomic write pattern</name>
  <files>~/.claude/scripts/registry-update.sh</files>
  <action>
    Enhance all registry-update functions with atomic writes:

    For every function that modifies registry:
    1. Create temp file: TEMP_REGISTRY="{REGISTRY}.tmp.$$"
    2. Copy current registry to temp: cp "$REGISTRY" "$TEMP_REGISTRY"
    3. Modify temp file using jq
    4. Validate JSON: jq empty "$TEMP_REGISTRY" || { rm "$TEMP_REGISTRY"; exit 1; }
    5. Create backup: cp "$REGISTRY" "${REGISTRY}.backup.$(date +%s)"
    6. Move temp to registry: mv "$TEMP_REGISTRY" "$REGISTRY"
    7. Call symlink-organize.sh if successful
    8. Return success/failure status

    Pattern:
    ```bash
    update_last_accessed() {
      local project_id="$1"
      local timestamp=$(date -u +%Y-%m-%dT%H:%M:%SZ)
      local temp_registry="${REGISTRY}.tmp.$$"

      cp "$REGISTRY" "$temp_registry" || return 1

      jq "(.projects[] | select(.id==\"$project_id\") | .last_accessed) = \"$timestamp\"" \
        "$temp_registry" > "$temp_registry.validated" || {
        rm -f "$temp_registry" "$temp_registry.validated"
        return 1
      }

      if jq empty "$temp_registry.validated" 2>/dev/null; then
        cp "$REGISTRY" "${REGISTRY}.backup.$(date +%s)"
        mv "$temp_registry.validated" "$REGISTRY"
        rm -f "$temp_registry"
        ~/.claude/scripts/symlink-organize.sh --quiet 2>/dev/null || true
        return 0
      else
        rm -f "$temp_registry" "$temp_registry.validated"
        return 1
      fi
    }
    ```

    Apply this pattern to all update functions:
    - update_last_accessed
    - toggle_favorite / set_favorite
    - set_category / set_description / set_status
    - increment_session_count
    - Any other modification function
  </action>
  <verify>

    3. Verify JSON is valid: jq . ~/.claude/projects/.registry.json

    5. Verify backup was created: ls -la ~/.claude/projects/.registry.json.backup.*
