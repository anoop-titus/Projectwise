**Phase goal from ROADMAP:** Migration script with rollback capability and backup.

**Dependency:** All phases 1-4 complete (all scripts and functions in place).

**What we're building:**
- migrate.sh script that:
  1. Checks if already migrated (idempotent)
  2. Creates backup of current .zshrc
  3. Initializes registry with existing projects
  4. Creates /active/ and /favorites/ symlinks
  5. Verifies all components working
  6. Provides rollback option if issues found
  7. Shows summary of changes made

- Rollback script that:
  1. Restores .zshrc from backup
  2. Optional: Remove registry/symlinks/scripts
  3. Verify rollback successful

<task type="auto">
  <name>Task 2: Create rollback.sh script</name>
  <files>~/.claude/scripts/rollback.sh</files>
  <action>
    Create rollback script for emergency use:

    1. Startup:
       - Check if migration was completed: test -f ~/.claude/migration-completed.txt
       - If not: "No migration found. Nothing to rollback."
       - Ask for confirmation: "Rollback will restore old .zshrc. Continue? (y/n)"

    2. Restore .zshrc:
       - Find most recent backup: ls -t ~/.claude/backups/zshrc-before-migration-*.sh | head -1
       - If not found: "Error: No backup found. Cannot rollback."
       - Restore: cp {backup} ~/.zshrc
       - Verify: source ~/.zshrc succeeds

    3. Optional: Remove new components
       - Ask: "Also remove new components? (registry, symlinks, scripts)"
       - If yes:
         * Remove: ~/.claude/projects/.registry.json
         * Remove: ~/.claude/projects/active/
         * Remove: ~/.claude/projects/favorites/
         * Note: Scripts are NOT removed (can be useful)

    4. Verify rollback:
       - source ~/.zshrc
       - type claude (verify old function loaded)
       - Check for errors

    5. Show summary:
       ```
       ✅ Rollback Complete!

       Restored:
       - ✅ .zshrc from backup

       Optional removal:
       - [ ] Registry (run manually if desired)
       - [ ] Symlink dirs
       - [ ] Scripts (kept for reference)

       To fully remove new system:
       rm -rf ~/.claude/projects/.registry.json ~/.claude/projects/active ~/.claude/projects/favorites

       Backups preserved:
       - ~/.claude/projects/.registry.json.pre-migration
       - Old .zshrc backups in ~/.claude/backups/

       Open new shell to test.
       ```

    6. Error handling:
       - If restore fails: "Error restoring .zshrc. Manual intervention needed."
       - Show backup location so user can restore manually
       - Don't delete anything if restore fails
  </action>
  <verify>
    Test rollback (after testing migration):
    1. Run: ~/.claude/scripts/rollback.sh
    2. Choose to remove components
    3. Verify: Registry removed, old .zshrc restored
    4. source ~/.zshrc succeeds
  </verify>
  <done>Rollback script fully implemented and tested</done>
</task>
