  <action>
    Create comprehensive user guide with sections:

    1. Introduction
       - What changed: Old numbered menu → New FZF selector
       - Why: Better discoverability, rich metadata, favorites, categories
       - Who benefits: Users with many projects

    2. Quick Start (5 minutes)
       - Run: claude (launch selector)
       - Type to search
       - Press Enter to select
       - Try: claude-status (see stats)

    3. Features Overview
       - FZF Selector: Fuzzy search, preview, keyboard navigation
       - Browse Mode: All folders, recursive navigation
       - Favorites: Star projects for quick access
       - Categories: Filter by type
       - Helper Commands: Project management without launching claude

    4. Keyboard Reference (detailed)
       - In Selector:
         * Type: Fuzzy search across all projects
         * Arrow Up/Down: Navigate list
         * Ctrl-F: Switch to Favorites mode
         * Ctrl-C: Switch to Categories mode
         * Ctrl-B: Browse ALL folders
         * Enter: Select project
         * Escape: Cancel and return to shell

    5. CLI Commands (detailed)
       - claude: Launch with selector (or --new for new project)
       - claude-favorite: Toggle favorite status
       - claude-info: Show project metadata
       - claude-list [mode]: Select project from list
       - claude-status: Show statistics

    6. Workflows (common tasks)
       - Create new project: claude --new
       - Open recent project: claude (select from recent)
       - Open favorite project: claude then Ctrl-F
       - Browse all folders: claude then Ctrl-B
       - Toggle favorite: claude-favorite
       - View project info: claude-info
       - See all projects: claude-status --detailed

    7. Metadata System
       - What's tracked: Name, description, tags, category, status, dates
       - Where stored: .registry.json (central) + PROJECT.json (per-project)
       - How to edit: via helper commands (claude-favorite, etc)
       - Manual editing: Edit .registry.json directly (use registry-recover.sh if corrupted)

    8. Troubleshooting
       - FZF not working: Check if fzf is installed
       - Selector shows wrong projects: Run registry-init.sh --force
       - Symlinks not updating: Check registry-update.sh output
       - Registry corrupted: Use registry-recover.sh to restore

    9. Advanced Topics
       - Editing registry directly: .registry.json structure
       - Custom categories: Add to .registry.json categories array
       - Project status: active/paused/archived meanings
       - Session tracking: Last accessed, session count tracking
       - Rollback: If issues, use ~/.claude/scripts/rollback.sh

    10. FAQ
        - Q: How do I add a new project?
          A: Use `claude --new` or manually copy folder to ~/.claude/projects/
        - Q: Can I edit project metadata?
          A: Use helper commands (claude-favorite) or edit .registry.json
        - Q: What if I break the registry?
          A: Use registry-recover.sh to restore from backup
        - Q: How do I hide archived projects?
          A: Change status to "archived" in registry
  </action>
