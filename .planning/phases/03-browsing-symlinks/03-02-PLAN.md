<task type="auto">
  <name>Task 2: Create symlink-organize.sh script</name>
  <files>~/.claude/scripts/symlink-organize.sh</files>
  <action>
    Create symlink-organize.sh script that:
    1. Reads .registry.json
    2. For each project with status="active":
       - Extract folder_name
       - Create symlink: ~/.claude/projects/active/{id} -> ../{folder_name}
       - If symlink already exists and points to same folder, skip
       - If symlink exists and points elsewhere, remove and recreate
    3. For each project with favorite=true:
       - Create symlink: ~/.claude/projects/favorites/{id} -> ../{folder_name}
       - Same logic as above
    4. Clean up stale symlinks:
       - Find symlinks in /active/ that don't correspond to any active project
       - Find symlinks in /favorites/ that don't correspond to any favorite project
       - Remove stale symlinks
    5. Log results:
       - "Created X symlinks in /active/"
       - "Created Y symlinks in /favorites/"
       - "Removed Z stale symlinks"
    6. Handle errors gracefully:
       - If registry doesn't exist, exit with message
       - If folder doesn't exist, skip (broken symlink cleanup will remove it)
       - If permission denied, skip with warning

    Make script idempotent (safe to run multiple times).
  </action>
  <verify>
    Call: ~/.claude/scripts/symlink-organize.sh

    Output should show:
    - "Created X symlinks in /active/"
    - "Created Y symlinks in /favorites/"
    - Number of created symlinks matches registry (active + favorite counts)
  </verify>
  <done>Script created, idempotent, handles errors gracefully</done>
</task>
